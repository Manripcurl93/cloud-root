# cloud-root
Spring Cloud 项目
 ### 模块
``` lua
cloud-root
├── cloud-auth    -- 授权服务提供[4000]
├── cloud-common  -- 系统公共模块
├── cloud-config  -- 配置中心[8888]
├── cloud-eureka  -- 服务注册与发现[8761]
├── cloud-gateway -- 网关[9000]
└── cloud-modules -- 微服务模块
      ├── daemon-service -- 调度中心[8020]
      └── upms-service   -- 权限管理[8080]
```
